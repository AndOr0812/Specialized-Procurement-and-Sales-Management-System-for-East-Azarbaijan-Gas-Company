# سیستم مدیریت خرید‌های داخلی امور‌ تدارکات و عملیات شرکت گاز استان آذربایجان شرقی

ساخته شده توسط [فریمورک تحت وب جنگو Django Web Framework](https://www.djangoproject.com)

# درباره این پروژه
این پروژه در حقیقت یک سیستم مدیریت خرید هست که با توجه به روال های اداری شرکت نفت فرم های خاص مربوطه طراحی شده و به کارمندان بخش امور مالی و انبار داری شرکت نفت و گاز میتونه مفید باشه.البته با توجه به اینکه این سیستم یه مثال از بیزینس مدلینگ یا طراحی منطق یه مجموعه میباشد میتوان با تغییر یا الگو برداری ازش سیستم های با منطق خاص مورد نیاز خودتونو باهاش بسازین.
این سیستم یه مثال از اینه که میشه با اوپن سورس کردن برنامه های اداری و دانشگاهی اونهارو درجهت بهبود و همه فهم بودن و شفافیت یاری کرد و اشتراک تکنولوژی مدرن تر اداری رو بین استانها و شهرها رو سرعت بخشید.

# How To:

## Installation:

    1. Download and extract zip file
    3. Download & install python or anaconda softwere on your Computer
    4. open cmd and run these commands:
        1. cd { extracted files folder directory 4 ex: C:\Downloads\Chortke\ }
        2. pip install -r requirments.txt
        3. python manage.py makemigrations
        4. python manage.py migrate
        5. python manage.py createsuperuser
        6. python manage.py runserver
        7. open this link: http://127.0.0.1:8000
